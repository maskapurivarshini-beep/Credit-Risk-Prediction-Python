# Credit-Risk-Prediction-Python
End-to-end credit risk prediction project using Python, featuring synthetic target engineering, EDA, and machine learning models (Logistic Regression, Random Forest, XGBoost, Neural Networks). Evaluates model performance using AUC-ROC on imbalanced data and delivers actionable insights for credit risk assessment.
